## 3. 掲載とクレジット 〔BY〕

- `BY` が付く場合、掲載時にクレジット表記を行う。
- 表記例：`Photo. [撮影者名] / Model. [モデル名]`
- SNS掲載時は、可能な範囲で本文内のメンションおよび画像内クレジットを行う。
