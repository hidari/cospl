# CosPL タグ仕様 v1.0

利用条件はタグの組み合わせで表します。

## 識別子の書式

```
CosPL <バージョン> / <タグをハイフンで連結>
```

例：

```
CosPL 1.0 / BY-NC-NAI-TD
CosPL 1.0 / BY-NC-TD-MR
```

## コアタグ

| タグ | 名称 | 内容 |
|------|------|------|
| `BY` | Attribution（クレジット表記） | 掲載時にクレジット表記が必要 |
| `NC` | NonCommercial（非商用） | 商用利用は不可。希望する場合は要相談 |
| `NAI` | No-AI（AI不可） | 生成AIの学習用データ・生成素材としての利用を禁止 |
| `TD` | Takedown（取り下げ可） | 公開後の取り下げ依頼に撮影者・モデル双方が応じる |

`BY` と `NC` は Creative Commons の同名タグと同じ趣旨です。CosPL 独自のタグは `NAI` と `TD` です。

## オプションタグ

| タグ | 名称 | 内容 |
|------|------|------|
| `MR` | Model-ROM（モデル本人の有償頒布可） | モデル本人がクレジット付きで自身の有償頒布物（ROM・写真集等）に使用することを許可 |

`NC`（非商用）を付けつつ `MR` を併記すると、「商用は原則不可だが、モデル本人の有償頒布だけは許可」という意味になります。

## 注意

- タグはあくまで条件の要約です。実際の許諾内容は本文（CosPL-1.0.md）およびテンプレートの記載が優先します。
- 新しいタグを独自に追加した場合は、その定義を併記してください。
